Java基础数据类型

Java语言提供了八种基本数据类型，分为四类：整数类型、浮点类型、字符类型和布尔类型。

1. 整数类型

数据类型 关键字 内存占用 默认值 取值范围
字节型 byte 1字节 0 -128 到 127
短整型 short 2字节 0 -32,768 到 32,767
整型 int 4字节 0 -2,147,483,648 到 2,147,483,647
长整型 long 8字节 0L -9,223,372,036,854,775,808 到 9,223,372,036,854,775,807

2. 浮点类型

数据类型 关键字 内存占用 默认值 描述
单精度 float 4字节 0.0f 单精度32位IEEE 754浮点数
双精度 double 8字节 0.0d 双精度64位IEEE 754浮点数

3. 字符类型

数据类型 关键字 内存占用 默认值 描述
字符型 char 2字节 ‘\u0000’ 表示一个16位Unicode字符

4. 布尔类型

数据类型 关键字 内存占用 默认值 描述
布尔型 boolean 不确定 false 表示真或假的值

注意：Java中布尔类型的大小没有明确规定，取决于JVM实现。
