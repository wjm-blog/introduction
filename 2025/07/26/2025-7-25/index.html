Java Hello World 示例

这是一个简单的 Java Hello World 程序，用于演示如何编写和运行基本的 Java 应用程序。

代码示例

代码说明

  1. public class HelloWorld：定义了一个名为 HelloWorld 的公共类。
  2. public static void main(String[] args)：这是程序的入口点，Java 虚拟机 (JVM) 会从此处开始执行程序。
  3. System.out.println("Hello, World!");：这行代码会在控制台输出 “Hello, World!”。

如何运行

  1. 将上述代码保存为 HelloWorld.java 文件。
  2. 打开命令行工具，导航到文件所在的目录。
  3. 使用 javac HelloWorld.java 命令编译 Java 文件。
  4. 使用 java HelloWorld 命令运行编译后的程序。

运行成功后，你将在控制台上看到输出：Hello, World!。
