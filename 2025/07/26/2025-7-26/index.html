Java 基础变量示例

这是一个简单的 Java 基础变量示例，用于演示如何声明和使用不同类型的变量。

代码示例

代码说明

  1. int age = 25;：声明一个整型变量 age 并赋值为 25。
  2. double height = 5.9;：声明一个双精度浮点型变量 height 并赋值为 5.9。
  3. char initial = 'J';：声明一个字符型变量 initial 并赋值为 ‘J’。
  4. boolean isStudent = true;：声明一个布尔型变量 isStudent 并赋值为 true。
  5. String name = "John Doe";：声明一个字符串变量 name 并赋值为 “John Doe”。
  6. System.out.println(...);：用于在控制台输出变量的值。

如何运行

  1. 将上述代码保存为 BasicVariables.java 文件。
  2. 打开命令行工具，导航到文件所在的目录。
  3. 使用 javac BasicVariables.java 命令编译 Java 文件。
  4. 使用 java BasicVariables 命令运行编译后的程序。

运行成功后，你将在控制台上看到输出的变量值。
